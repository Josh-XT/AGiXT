# Anthropic Claude

- [Anthropic Claude](https://console.anthropic.com/docs/access)
- [AGiXT](https://github.com/Josh-XT/AGiXT)

## Quick Start Guide

- Get your Anthropic API key at <https://console.anthropic.com/settings/keys>.

### Update your agent settings

1. Set `AI_PROVIDER` to `claude`.
2. Set `ANTHROPIC_API_KEY` to your API key.
3. Set `ANTHROPIC_MODEL` to `claude-sonnet-4-20250514`. List of models available at <https://docs.anthropic.com/claude/docs/models-overview>.
4. Set `ANTHROPIC_MAX_TOKENS` to the maximum number of input tokens. Default is `200000` for Claude.
